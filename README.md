# Etch-A-Sketch

A digital recreation of the classic Etch-A-Sketch toy, built with HTML, CSS, and JavaScript.

![Etch-A-Sketch Screenshot](screenshot.png)

## Features

- Responsive drawing grid
- Customizable grid size
- Multiple modes:
  - Classic (Gray)
  - Rainbow
  - White (Ereaser)
